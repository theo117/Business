# Deployment Checklist

Use this after publishing the updated site on `https://www.teodordev.co.za`.

## Core Checks

1. Confirm the homepage loads at:
   - `https://www.teodordev.co.za/`
2. Confirm the pricing page loads at:
   - `https://www.teodordev.co.za/price.html`
3. Confirm the thank-you page loads at:
   - `https://www.teodordev.co.za/thank-you.html`
4. Confirm the monthly-plan guide loads at:
   - `https://www.teodordev.co.za/monthly-plan-guide.html`
5. Confirm the 404 page loads when configured by hosting:
   - `https://www.teodordev.co.za/404.html`

## SEO Checks

1. Confirm `robots.txt` loads:
   - `https://www.teodordev.co.za/robots.txt`
2. Confirm `sitemap.xml` loads:
   - `https://www.teodordev.co.za/sitemap.xml`
3. Confirm canonical tags use `https://www.teodordev.co.za`
4. Confirm Open Graph and Twitter image URLs use the new domain
5. Confirm structured data references the new domain and `Teodor Dev`

## Redirect Checks

1. Set a permanent `301` redirect from:
   - `https://www.theodorenelson.co.za`
   to:
   - `https://www.teodordev.co.za`
2. Test old URLs and confirm they land on the correct new URLs

## Search Console Checks

1. Add and verify `https://www.teodordev.co.za`
2. Submit:
   - `https://www.teodordev.co.za/sitemap.xml`
3. If keeping the old domain, monitor it for redirect and indexing issues

## Form Checks

1. Submit one live enquiry from `price.html`
2. Confirm it redirects to `thank-you.html`
3. Confirm the Web3Forms email arrives
4. Confirm monthly-plan leads can continue to:
   - `monthly-plan-guide.html`
   - the Google qualification form

## Brand Consistency Checks

1. Confirm public pages show `Teodor Dev`
2. Confirm logo alt text says `Teodor Dev logo`
3. Confirm footer branding is consistent
4. Confirm titles and descriptions reflect `Teodor Dev`

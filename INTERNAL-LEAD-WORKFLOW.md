# Internal Lead Workflow

This document is for internal use only.

It is intentionally not linked from the public website and should be treated as a private operating note for handling enquiries.

## Purpose

Keep the lead-handling process consistent for:

- once-off build enquiries
- monthly-plan enquiries
- enquiries where the client is not yet sure

## Public-Facing Assets Already In Use

- `price.html`
- `monthly-plan-guide.html`
- `thank-you.html`
- Google qualification form
- `ENQUIRY-RESPONSE-TEMPLATES.md`

## Internal Workflow

### 1. New Enquiry Arrives

Check:

- client name
- client email
- `project_model`
- project brief

## 2. If Project Model = Once-Off Build

Use the `Once-Off Build Enquiry` template from `ENQUIRY-RESPONSE-TEMPLATES.md`.

Then:

1. Review likely page count
2. Review whether the client needs WordPress or a simpler build
3. Decide whether to request more detail before quoting

## 3. If Project Model = Monthly Plan

Use the `Monthly Plan Enquiry` template from `ENQUIRY-RESPONSE-TEMPLATES.md`.

Then:

1. Send the monthly-plan overview
2. Send the Google qualification form
3. Wait for qualification form completion
4. Review the answers before preparing any quote

## 4. If Project Model = Not Sure Yet

Use the `Not Sure Yet` template from `ENQUIRY-RESPONSE-TEMPLATES.md`.

Then:

1. Review the project brief
2. Decide whether it sounds like:
   - a straightforward once-off build
   - a larger managed project
3. Recommend the most suitable path in your follow-up

## 5. Monthly-Plan Review Checklist

Before preparing a monthly-plan quote, confirm:

1. likely site size
2. required features
3. content readiness
4. whether ongoing updates are expected
5. preferred platform
6. support expectations
7. whether the scope is stable enough to quote

## 6. Good Rule For Monthly Plans

Do not quote a monthly-plan arrangement until:

- the initial enquiry has been read
- the qualification form has been completed
- the scope looks clear enough to structure properly

## 7. Practical Admin Habit

Keep a simple status for each lead:

- `new`
- `replied`
- `waiting for qualification`
- `ready to quote`
- `quoted`
- `closed`

# SEO Checklist

Use this after deployment to strengthen branded search for `Teodor` and `Teodor Web Solutions`.

## Technical

- Confirm the live site resolves at `https://www.theodorenelson.co.za/`.
- Check that `https://www.theodorenelson.co.za/robots.txt` loads.
- Check that `https://www.theodorenelson.co.za/sitemap.xml` loads.
- Make sure only one domain version is primary. Redirect non-`www` to `www` if `www` is the canonical version.
- Verify every main page returns `200 OK`.
- Keep HTTPS enabled on the whole site.

## Google

- Add the property in Google Search Console.
- Submit the sitemap: `https://www.theodorenelson.co.za/sitemap.xml`.
- Request indexing for the homepage, projects page, and pricing page.
- Check Search Console coverage for crawl or canonical issues.

## Brand Signals

- Use `Teodor Web Solutions` consistently on LinkedIn, Facebook, Instagram, and any business directory.
- If available, link those profiles back to the website.
- Ask clients or collaborators to mention `Teodor Web Solutions` when linking to your site.
- Publish a Google Business Profile if the business qualifies and the published business details are accurate.

## Content

- Add an About section or page focused on `Teodor Web Solutions`.
- Add a portfolio description for each project with a few sentences of original text.
- Add a short FAQ section about services, pricing, timelines, and support.
- Keep publishing pages or case studies that naturally mention `Teodor Web Solutions`.

## Stronger Ranking For `teodor`

- Ranking for the single word `teodor` is harder than ranking for `Teodor Web Solutions`.
- First aim to rank for `Teodor Web Solutions`, `Teodor web design`, and `Teodor Web Solutions South Africa`.
- A brand-matching domain such as `teodorwebsolutions.co.za` or similar would strengthen the signal further if you ever decide to change domains.
